# Security Policy

## Reporting a vulnerability

Please report vulnerabilities privately.

- Preferred: https://github.com/vdkit/VDK-Blueprints/security/policy
- Contact: security@vdk.tools

Include:

1. Affected repository/repositories
2. Reproduction steps or proof of concept
3. Potential impact
4. Suggested mitigation (if available)

We will triage reports as quickly as possible and coordinate a responsible fix and disclosure.
